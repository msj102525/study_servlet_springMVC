# Servlet 프로젝트

이 프로젝트는 순수 Servlet 기반 웹 애플리케이션 예제입니다.

## 실행 방법

1. JDK 11 이상이 설치되어 있어야 합니다.
2. Gradle 빌드 도구를 사용합니다.
3. 아래 명령어로 실행할 수 있습니다.

```sh
./gradlew bootRun
```

## 주요 기능

- 회원 가입, 회원 목록 조회
- JSP 기반 View 렌더링
- 서블릿과 JSP를 활용한 MVC 패턴 예제
